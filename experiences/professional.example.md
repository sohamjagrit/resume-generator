# Professional Experience

The Setup wizard fills this file from your uploaded resumes; you confirm before
it's saved. Each `## Company` block becomes one entry in the generated resume.

Tag lines (`**Domain tags:**`, `**Skill tags:**`) feed the project/JD scorer.
Bullets must be factually accurate — Claude is forbidden from inventing tools
or metrics, so anything outside this file will never appear in a generated
resume.
